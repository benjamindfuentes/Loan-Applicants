# Loan Applicants Data
## 💼  Overview
An ongoing project analyzing and visualizing loan application data using SQL. 

I'll be tracking/analyzing loan applications submitted for 6 loan purposes: 
1. Education, 
2. Medical, 
3. Venture,
4. Personal
5. Debt Consolidation
6. Home Improvement
